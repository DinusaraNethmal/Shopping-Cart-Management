[cite_start]This project is a fully functional web-based shopping mall designed to facilitate e-commerce operations[cite: 2]. It is built using **Java Spring Boot**, **MySQL**, and a dynamic **HTML/CSS/JavaScript** frontend.

[cite_start]The system implements six core modules covering the entire buying cycle, including User Management, Product Catalog, Shopping Cart, Order Processing, Category/Discount Management, and Customer Reviews[cite: 377]. [cite_start]It supports role-based access for Customers (to browse and buy) and Administrators (to manage inventory and orders)[cite: 384].
