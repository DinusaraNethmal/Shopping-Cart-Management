# E-Commerce Shopping Cart System

## ðŸ“– Project Overview
This project is a fully functional web-based shopping mall designed to facilitate e-commerce operations. It is built using **Java Spring Boot**, **MySQL**, and a dynamic **HTML/CSS/JavaScript** frontend.

The system implements six core modules covering the entire buying cycle, including User Management, Product Catalog, Shopping Cart, Order Processing, Category/Discount Management, and Customer Reviews. It supports role-based access for Customers (to browse and buy) and Administrators (to manage inventory and orders).
